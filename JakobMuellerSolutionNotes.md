# Notizen für Lösungsansatz

#### Mögliche Interaktion

<ol>
  <li>Textfeld Nutzen -> TN</li>
  <li>Button Klicken -> BK</li>
</ol>

#### Mögliche Ergebnisse

<ol>
  <li>Button Name Gleich -> BNG</li>
  <li>Button Name Anders -> BNA</li>
</ol>

## Schritte und folgendes Ergebnis
<ul>
  <li>TN TN = BNG</li>
  <li>BK BK = BNG</li>
  <li>BK TN = BNG</li>
  <li>TN BK = BNA</li>
</ul>
